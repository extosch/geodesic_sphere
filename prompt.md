Baue eine Webseite, die eine geodätische Kugel beliebiger Frequenz (F=1 ist Ikosaeder) 
konstruiert und visualisiert, variabel als Flächen oder Gittermodell. 
Der Algorithmus unterteilt sukzessive die gleichseitigen Dreiecke des Ikosaeders
durch Unterteilung der Kanten in 2, 3, 4, 5, .. Abschnitte, die dann die Kanten der
neuen Dreiecke der um 1 erhöhten Frequenz bilden.
  
Design: Schwarzer Hintergrund, Flächen Grau, Gitter wie, Licht von schräg rechts oben.
Die Webseite beinhaltet ein Eingabefeld für die Frequenz F und ein Ausgabefeld für die Anzahl der Ecken, Kanten,
Dreiecke und Zahl der unterschiedlichen Kantenlängen und zwei Checkboxen für Flächen und Gitter-Modell.
Das erzeugte Objekt rotiert um die eigeen Achse. Der Maus werden die üblichen Steuerfunktionen Zoom, Pan und
Rotation (um Objekt-Achse) zugeordnet.

Du bist ein erfahrender Programmierer und lässt dir Zeit zum Nachdenken.
Technik: html, css, Javascript +  WEbGL.
Erzeuge separate index, script, style und README files

## English Translation

Build a website that constructs and visualizes a geodesic sphere of arbitrary frequency (F=1 is icosahedron), 
variable as faces or wireframe model.
The algorithm successively subdivides the equilateral triangles of the icosahedron
by dividing the edges into 2, 3, 4, 5, .. sections, which then form the edges of
the new triangles of the frequency increased by 1.

Design: Black background, faces gray, wireframe white, light from upper right diagonal.
The website contains an input field for frequency F and an output field for the number of vertices, edges,
triangles and number of different edge lengths and two checkboxes for face and wireframe model.
The generated object rotates around its own axis. The mouse is assigned the usual control functions zoom, pan and
rotation (around object axis).

You are an experienced programmer and take your time to think.
Technology: html, css, Javascript + WebGL.
Create separate index, script, style and README files
